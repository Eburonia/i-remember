About iRemember

In many countries, students use the flashcard system to learn another language. For example, Dutch students learn English words by writing down the Dutch word on the front and the English translation on the backside of a flashcard.

By reading the Dutch word on the frontside (question), they try to guess the English translation on the backside (answer). When repeated over a certain amount of time, you slowly start to remember the answer to the question, which is the purpose of the flashcard system.

The iRemember website will help language learners remember words or sentences in a different language. It can also help remember other kinds of information, for example, the capital cities of certain countries. The website uses the flashcard system as described above.

The flashcard system combined with the spaced repetition system will help you learn a language faster.

Use the website on both desktop or smaller devices (iPad or iPhone).

We recommend reading our manual on the 'How to Use' page first before creating flashcard sets or start using the 'Practice' page.
